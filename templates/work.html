PROJECT 2. "Images"

LINK project2.nartovcv.online

Description


Pretty small project, but it gives a possibility to work with a lot of technologies and python libraries. In this project user can load and store original images and thumbnails. Also user has a possibility to create expiry link to every image he has access. Access to images defines by user's account tier. You can create different account tiers and thumbnail sizes as superuser via admin panel. This project contains 2 parts: simple user interface by link project2.nartovcv.online/register and also REST API part, which is available after registration by links project2.nartovcv.online/api/v1/images/, project2.nartovcv.online/api/v1/upload/

Technologies


  • Django
  • Django Rest Framework
  • PostgreSQL
  • Celery
  • Redis
  • Unit Tests
  • Docker
  • HTML
  • CSS
  • Bootstrap
  • GIT
  • Buying and setting up real production server and SSL Certificate
  • Project was deployed on real production server using Nginx and Gunicorn

GIT


https://github.com/AndriiNartov/image_project.git
